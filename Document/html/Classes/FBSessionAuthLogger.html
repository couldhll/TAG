addExtrasForNextEvent:

@abstract Add JSON-serializable data to the ‘extras’ JSON blob that is attached to these auth events. The keys and values passed in here will be attached to the next event that is logged via this logger, and cleared after that. They are lost if no events are logged after a call to this method. Multiple calls to this method between log events, will append all the extras together.
