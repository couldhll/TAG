Overview

@class FBDynamicFrameworkLoader

@abstract This class provides a way to load constants and methods from Apple Frameworks in a dynamic fashion. It allows the SDK to be just dragged into a project without having to specify additional frameworks to link against. It is an internal class and not to be used by 3rd party developers.

This class is a generic class for loading Classes, NSStrings, and SecRandomRef. As new types are needed, they should be added and strongly typed.
