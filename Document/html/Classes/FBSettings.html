publishInstall:withHandler:

@method

+ (void)publishInstall:(NSString *)appID withHandler:(FBInstallResponseDataHandler)handler

Parameters

appID

A specific appID to publish an install for. If nil, uses [FBSession defaultAppID].

handler

A block to call with the server’s response.

Discussion

@abstract Manually publish an attributed install to the Facebook graph, and return the server response back in the supplied handler. Calling this method will implicitly turn off auto-publish. This method acquires the current attribution id from the facebook application, queries the graph API to determine if the application has install attribution enabled, publishes the id, and records success to avoid reporting more than once.
