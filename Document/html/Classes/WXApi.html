registerApp:

@brief WXApi的成员函数，向微信终端程序注册第三方应用。

+ (BOOL)registerApp:(NSString *)appid

Parameters

appid

微信开发者ID

Return Value

成功返回YES，失败返回NO。

Discussion

需要在每次启动第三方应用程序时调用。第一次调用后，会在微信的可用应用列表中出现。 iOS7及以上系统需要调起一次微信才会出现在微信的可用应用列表中。 @attention 请保证在主线程中调用此函数

sendReq:

@brief 发送请求到微信，等待微信返回onResp

+ (BOOL)sendReq:(BaseReq *)req

Parameters

req

具体的发送请求，在调用函数后，请自己释放。

Return Value

成功返回YES，失败返回NO。

Discussion

函数调用后，会切换到微信的界面。第三方应用程序等待微信返回onResp。微信在异步处理完成后一定会调用onResp。支持以下类型 SendAuthReq、SendMessageToWXReq等。

Declared In

WXApi.h

sendResp:

@brief 收到微信onReq的请求，发送对应的应答给微信，并切换到微信界面

+ (BOOL)sendResp:(BaseResp *)resp

Parameters

resp

具体的应答内容，调用函数后，请自己释放

Return Value

成功返回YES，失败返回NO。

Discussion

函数调用后，会切换到微信的界面。第三方应用程序收到微信onReq的请求，异步处理该请求，完成后必须调用该函数。可能发送的相应有 GetMessageFromWXResp、ShowMessageFromWXResp等。
