Inherits from PFObject : NSObject
Conforms to PFSubclassing
Declared in PFProduct.h

Overview

Represents an in-app purchase product on the Parse server. By default, products can only be created via the data browser; saving a PFProduct will result in error. However, the products' metadata information can be queried and viewed.

This class is currently for iOS only.
