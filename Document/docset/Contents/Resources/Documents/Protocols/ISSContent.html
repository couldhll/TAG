addFlickrUnitWithPhoto:title:description:tags:isPublic:isFriend:isFamily:safetyLevel:contentType:hidden:

@brief  添加Flickr内容单元
- (void)addFlickrUnitWithPhoto:(id<ISSCAttachment>)photo title:(NSString *)title description:(NSString *)description tags:(NSString *)tags isPublic:(NSNumber *)isPublic isFriend:(NSNumber *)isFriend isFamily:(NSNumber *)isFamily safetyLevel:(NSNumber *)safetyLevel contentType:(NSNumber *)contentType hidden:(NSNumber *)hidden

Parameters

photo

图片对象

title

标题

description

相片描述。可能包含某受限的 HTML。

tags

適用於相片的以空格分隔的標籤清單。

isPublic

設定為 0 表示否，1 表示是。指定誰可以檢視相片。

isFriend

設定為 0 表示否，1 表示是。指定誰可以檢視相片。

isFamily

設定為 0 表示否，1 表示是。指定誰可以檢視相片。

safetyLevel

設置 1 為「安全級」、2 為「輔導級」、3 為「限制級」。

contentType

設置 1 為「相片」、2 為「螢幕截圖」、3 為「其他」。

hidden

設置 1 為「相片」、2 為「螢幕截圖」、3 為「其他」。

Parameters

title

标题

url

链接地址
