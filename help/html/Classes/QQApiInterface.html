getQQInstallUrl

获取QQ下载地址

+ (NSString *)getQQInstallUrl

Return Value

iPhoneQQ下载地址

Discussion

如果App通过QQApiInterface#isQQInstalled和QQApiInterface#isQQSupportApi检测发现QQ没安装或当前版本QQ不支持API调用，可引导用户通过打开此链接下载最新版QQ。
